# ![Origin Protocol](origin-header.png)

A decentralized e-commerce store served entirely from IPFS.

## Repo structure

- shop: front-end code
- backend: server code
- packages: utility packages
- scripts: development helper scripts

## Getting started

To bring up the example store and server on your local:

    yarn install
    DATA_DIR=example yarn run start
